Este projeto ainda só funciona em dispositivos Android, ele não foi utilizado nem validado em dispositivos com iOS.

Para que essas instruções no repositório sejam bem-sucedidas, habilite primeiro as opções de depuração USB no seu dispositivo. Após habilitado, você pode prosseguir.


ATENÇÃO: tome cuidado com o caminho onde você direciona os arquivos. De backup e os criptografados. Neste script .bash feito, os caminho foram direcionados baseados em caminhos do sistema operacional Fedora baseado em Linux.
